About Me

Geomatics Engineer with a builder's mindset.

I have hands-on experience across the full geospatial stack, from satellite image classification and hydrological modelling to deploying full-stack web mapping applications. My foundation in Geomatics Engineering gives me a technical understanding of spatial data that informs every project.

On the development side, I build and ship production-ready applications using Mapbox GL JS, Supabase, and Vercel. I care about the end user's experience. A map should be fast, readable on mobile, and actually answer the question it was built to answer.

I'm based in Abuja, Nigeria and work remotely with clients across different regions and time zones.

How I Work

My Approach

Every project follows a clear process, from understanding the spatial problem to shipping a maintainable, documented solution.

01
Understand

I start by understanding the spatial problem, the data available, and what the end user actually needs from the output: map, model, or analysis.

02
Design

I plan the data pipeline, tool stack, and interface, whether that is a Mapbox web app, an ArcGIS workflow, or a Revit BIM model.

03
Build

I build iteratively, keeping clients in the loop with staging deployments on Vercel or intermediate GIS outputs for review.

04
Deliver

I deliver fully documented, production-ready work: deployed applications, structured GIS outputs, or BIM models ready for the next stage.
